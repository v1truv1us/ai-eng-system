# Welcome to ai-eng-system

Advanced engineering toolkit with 28 specialized agents, 17 commands, and 8 skills for Claude Code and OpenCode platforms.

## Quick Start

```bash
# Install globally
npm install -g ai-eng-system

# Run installation script
npm run install:global

# Verify installation
bun run build
```
