# Welcome to ai-eng-system

AI engineering toolkit with 32 agents, 42 commands, and 12 skills for Claude Code and OpenCode.

## Quick Start

```bash
npm install -g @ai-eng-system/cli
ai-eng install --scope project
```

## Package Model

- `@ai-eng-system/core` - library and content-loading helpers
- `@ai-eng-system/toolkit` - generated Claude/OpenCode/plugin assets
- `@ai-eng-system/cli` - executable installer and workflow package

## Core Workflow

- `/ai-eng/research`
- `/ai-eng/specify`
- `/ai-eng/plan`
- `/ai-eng/work`
- `/ai-eng/review`
- `/ai-eng/simplify`
