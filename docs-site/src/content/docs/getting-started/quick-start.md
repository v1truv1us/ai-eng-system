# Quick Start

Get started with ai-eng-system in four steps.

## 1. Install the CLI

```bash
npm install -g @ai-eng-system/cli
```

## 2. Install project assets

```bash
ai-eng install --scope project
```

## 3. Verify setup

```bash
ai-eng --help
bun run build
```

## 4. Use the workflow

Start with:

- `/ai-eng/research`
- `/ai-eng/specify`
- `/ai-eng/plan`
- `/ai-eng/work`
- `/ai-eng/review`

Useful follow-ups:
- `/ai-eng/ralph-wiggum`
- `/ai-eng/simplify`
- `/ai-eng/context`
- `/ai-eng/optimize`
