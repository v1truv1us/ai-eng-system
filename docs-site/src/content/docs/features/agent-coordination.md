# Agent Coordination

Three-mode workflow for specialized agent delegation.

---

## Plan Mode

Read-only research and planning

Gather context, analyze patterns

Create detailed implementation plans

---

## Build Mode

Implements changes with quality gates

Writes code following architectural guidelines

Routes to appropriate specialist agents

---

## Review Mode

Multi-perspective code review

Quality assessment and recommendations

No code modifications, read-only analysis

---
