# Session Commands

Toggle settings during active session.

---

## Toggle Auto-Approve

`/optimize-auto on` - Skip approval steps

`/optimize-auto off` - Require approval

Default: off (interactive mode)

---

## Set Verbosity

`/optimize-verbosity quiet` - Minimal output

`/optimize-verbosity normal` - Balanced detail (default)

`/optimize-verbosity verbose` - Maximum detail

---

## Usage Example

Session starts with default settings

Toggle auto-approve for batch operations

Adjust verbosity based on context

Changes reset on new session
