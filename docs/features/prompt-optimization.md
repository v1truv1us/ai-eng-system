# Prompt Optimization

Automatic prompt enhancement using research-backed techniques.

---

## Automatic Optimization

Every prompt optimized by default

Applies expert persona, reasoning, stakes, self-evaluation

+45-115% quality improvement (peer-reviewed research)

---

## Escape Hatch

Prefix with `!` to bypass optimization

Example: `!just say hello`

Works in both Claude Code and OpenCode

---

## Step-by-Step Review

Use `/ai-eng/optimize "prompt"` for interactive flow

See each optimization step with details

Approve all, approve specific, modify, or cancel

---

## Techniques Applied

Expert persona: Detailed role with years of experience

Step-by-step reasoning: Systematic analysis

Stakes language: Importance and consequence framing

Self-evaluation: Confidence rating and assumptions

---

## Configuration

Configure behavior in `.claude/ai-eng-config.json`

Set verbosity: `quiet|normal|verbose`

Toggle auto-approve: `/optimize-auto on|off`

Enable/disable optimization in config
