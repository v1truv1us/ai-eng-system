# Knowledge

Static-first knowledge architecture lives here.

## Canonical files

- `index.md` - top-level map
- `domains/` - domain boundaries and concepts
- `entries/` - durable knowledge entries
- `hypotheses.md` - open claims to validate
- `rules.md` - stable operating rules

## Templates

- `templates/knowledge/index.md`
- `templates/knowledge/domain.md`
- `templates/knowledge/knowledge.md`
- `templates/knowledge/hypotheses.md`
- `templates/knowledge/rules.md`

Keep this folder file-backed and human-readable.
