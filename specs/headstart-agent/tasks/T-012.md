# T-012: Implement Scheduler and Job Processor

**Depends On:** T-005, T-009, T-010, T-011
**Priority:** critical
**Estimated Time:** 45 min
**Complexity:** Medium

## Acceptance Criteria
- [ ] `src/scheduler.ts` exports `startScheduler()` which registers a cron job using `config.SCHEDULE_MONTHLY`
- [ ] Cron job creates a queue job with `type='monthly-audit'` and empty payload
- [ ] `src/index.ts` starts: HTTP server on port 3000, scheduler, and job processor loop
- [ ] Job processor loop: polls `claimJob()` every 5 seconds, executes task via task registry, calls `completeJob` or `failJob`
- [ ] On task failure: calls `failJob`; if permanently failed, calls `reportError` and `sendAlertEmail`
- [ ] Graceful shutdown: on SIGTERM, finishes current job, closes server, exits
- [ ] Logs every state transition: job claimed, task started, completed, failed

## Files
- `src/scheduler.ts` — Create cron scheduler
- `src/index.ts` — Wire together server, scheduler, and job processor
