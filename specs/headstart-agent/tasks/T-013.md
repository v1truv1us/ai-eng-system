# T-013: Create Dockerfile and Local Dev Stack

**Depends On:** T-001, T-012
**Priority:** high
**Estimated Time:** 30 min
**Complexity:** Low

## Acceptance Criteria
- [ ] `Dockerfile` uses `oven/bun:1.2-alpine` base image
- [ ] Multi-stage or single-stage build that installs deps, builds, and runs with `bun run start`
- [ ] Exposes port 3000
- [ ] Defines volumes `/data` and `/repos`
- [ ] Image builds successfully: `docker build -t headstart-agent .`
- [ ] `docker-compose.yml` provides local dev stack with env file support
- [ ] `docker-compose up` starts the service and responds to `/health`

## Files
- `Dockerfile` — Create production Docker image
- `docker-compose.yml` — Create local development stack
