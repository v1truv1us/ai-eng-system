# Ralph Loop hooks (Cursor)

Shell hooks from [cursor/plugins/ralph-loop](https://github.com/cursor/plugins) (MIT).

Copied into marketplace plugins and the monolithic Cursor bundle during `bun run build`.

- `capture-response.sh` — detects completion promise in agent output
- `stop-hook.sh` — continues or ends the Ralph iteration loop
- `hooks.json` — Cursor hook manifest (installed as `cursor-hooks.json` when co-located with Claude hooks)
