# Platform Packaging Decisions

Date: 2026-05-23

## Chosen variation

**Variation A — Toolkit extension**

- `@ai-eng-system/toolkit` ships `.claude-plugin/`, `.opencode/`, `.cursor-plugin/`, `.gemini/`, `.pi/`, and `plugins/ai-eng-*`
- Pi distribution is build output at `dist/.pi/` synced into toolkit (no separate `@ai-eng-system/pi` package)
- No separate `@ai-eng-system/cursor` or `@ai-eng-system/gemini` packages in this iteration

## Rules mapping

- Cursor `.mdc` rules from `cursor-team-kit` copied to canonical `rules/cursor/`
- Build copies `rules/cursor/` → `dist/.cursor-plugin/rules/`
- Language rule packs under `rules/common/`, `rules/typescript/`, etc. remain separate; not copied to Cursor output

## Pi package consolidation (2026-05-23)

The standalone `@ai-eng-system/pi` package was removed. Pi assets are:

1. Generated by `buildPi()` → `dist/.pi/`
2. Synced by `build-toolkit.ts` → `packages/toolkit/.pi/`
3. Consumed via `pi install npm:@ai-eng-system/toolkit` using the toolkit `pi` manifest

Migration: replace `pi install npm:@ai-eng-system/pi` with `pi install npm:@ai-eng-system/toolkit`.
